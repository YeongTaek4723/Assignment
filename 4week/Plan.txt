사이트 목적 : 게임 리뷰/ 평가
사이트 이름 : Games Spot
구성원 : 오영택 (1인)


기능들

메인 페이지

- 왼쪽 상단 또는 Home를 누르면 처음 메인 화면으로 돌아간다. 

- 로그인 : 로그인 버튼을 누르면 로그인 화면으로 넘어간다.

- 검색 : 원하는 게임명을 입력하면 그 게임소개 창으로 넘어간다. 

- 컬렉션 : 일부 컬렉션 제목에 맞는 게임들을 모아놓은 패키지와 같다. 
  	이미지나 텍스트를 누르면 링크를 따라 들어간다.

- 추천 게임 : 추천 게임들을 보여준다.(총 5개)
                나머지는 모두보기를 만들어서 다른 창으로 넘어간다.


게임 소개 창 

- 게임 이미지 / 게임 제목 / 해당 게임 평점등이 보이고 
하단에는 그 게임에 대한 리뷰를 작성할 수 있으며 다른 유저가 작성한 리뷰가 보이도록 제작



로그인 
- 로그인을 하거나 회원가입 또는 아이디/ 비밀번호 찾기를 할 수 있다.

아이디/ 비밀번호 찾기 : 로그인 창이 아닌 다른 창으로 넘어가서 해당 아이디나 비밀번호를 찾을 수 있다.

회원가입 : 로그인 창이 아닌 다른 창으로 넘어가서 원하는 계정을 만들 수 있다.
